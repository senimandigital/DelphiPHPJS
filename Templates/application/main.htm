Selamat datang di DelphiPHPJS

Jika ingin membuat Aplikasi Desktop Opensource yang perlu kita lakukan hanyalah membuat sebuah Form dan itu terserah mau mengunakan Basic, C, Java, Pascal, Python itu semua bisa sekali. Kita hanya perlu Membuat satu Form yang berisi 1 Control WebBrowser kemudian mengcompile program tersebut, dengan demikian Main Aplikasi tidak akan mencapai 100 Kb kecuali untuk Delphi ukuran file exe yang dihasilkan mencapai 400 Kb, tapi itu masih terhitung sangat kecil jika dibandingan dengam membuat aplikasi desktop dengan NodeJS yang mencapai lebih dari 50Mb untuk main aplikasi meski hanya membuat aplikasi "Hello World".

Aplikasi yang sedang anda buka sekarang kami beri nama DelphiPHPJS, diberi nama seperti itu karena aplikasi ini mampu menjalankan Printah dari Delphi, PHP, Javascript dan VBScript sekaligus, tidak hanya itu karena termasuk disini adalah Html dan CSS tentunya. aplikasi ini menggunakan Browser yang sudah terinstal pada komputer Anda. itu yang membuatnya menjadi sangat ringan.

Kami berbeda dari pengelola Opensource lain, yang mungkin aktif melakukan update Produk-produk mereka, sedangkan kami memilih langkah lain, kami menyediakan teknologi pengembangan untuk setiap Opensource yang kami tulis. Update akan otomatis dilakukan ketika kami memutakhirkan teknologi yang kami miliki atau saat seseorang memicu update.

Apabila anda juga tertarik mengembangkan aplikasi desktop berbasis PHP anda bisa membuat folder dimana anda meletakan file ini sebagai folder php, untuk melakukan-nya anda bisa melakukan itu dengan menyertakan php portable, tutorial untuk membuat webserver php portabel banyak tersedia di internet, bahkan beberapa paket webserver sudah menyertakan fasilitas untuk membuat php portable.

Programmer Modern

Perkembangen teknologi sekarang begitu pesat, dan arah-nya semakin jelas. Diantara semua perkembangan teknologi pemrograman dan bahasa pemrograman, perkembangan Browser adalah yang tercepat dengan pengguna yang terus bertumbuh. HTML, CSS dan Javascript mendominasi diberbagai bidang. Diantara semua bahasa pemrograman dan model distribusi data HTML, CSS dan Javascript adalah standar yang harus kita kuasai dengan baik. Menguasai HTML, CSS dan javascript tidak akan pernah anda sesali sebagaimana anda mungkin akan menyesal telah menghabiskan waktu untuk mempelajari bahasa pemrograman yang lain.

Semua bahasa pemrograman visual desktop dan mobile mendukung penggunaan Control WebBrowser dan kondisi ini tidak terbantah-kan, kita bisa saja mendesain aplikasi dengan Control bawan yang sudah disediakan IDE dari Visual Basic, Delphi, Lazarus, Free Pascal, Net Beans atau yang lain-nya dan itu sah-sah saja. Tapi untuk saat ini belajar dan menggunakan Control bawaan dari Tool IDE sama sekali bukan lagi keputusan bijak, tidak bijak karena Control Bawaan tool IDE bersipat mengikat dan terbatas.

Bandingkan jika anda hanya menggunakan Control WebBrowser, kemudian menggunakan jQuery untuk mendesain Form kita akan dapat dengan mudah mendownload berbagai plugins jQuery dari internet misal jsTree, DataTables tidak hanya plugins jQuery karena kita juga pasti-nya bisa menggunakan plugins javascript seperti TinyMCE, MXGraph dan masih banyak lagi plugins mutakhir lain-nya.
